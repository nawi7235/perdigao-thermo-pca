# Thermodynamic variability at the Perdigao West profiler site

This project analyzes near-surface thermodynamic variability at the Perdigao West profiler site using ISFS observations. The analysis focuses on identifying dominant modes of variability using principal component analysis (PCA) and composite analysis.

## Research question
What are the dominant modes of thermodynamic variability at the site, and how do they differ between daytime and nocturnal conditions?

## Methods
- Derived thermodynamic variables from temperature, humidity, and pressure
- Applied PCA to identify dominant modes of variability
- Computed diurnal and extreme-value composites
- Performed additional nocturnal analysis to isolate stable boundary layer behavior

## Status
Initial analysis and repository structure completed. Additional interpretation and refinement of nocturnal processes will be added in the final submission.
